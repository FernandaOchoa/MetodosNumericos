# Serie de MacLaurin
Aplicación Móvil para calcular la Serie de MacLaurin para las funciones **Sen (x), Cos (x) y e (x)** 

### Instalación
El ejecutable se encuentra en la carpeta Ejecutable

   SerieMacLaurin  
----| Entregables  
------| Ejecutable
